String

STRING is a powerful protein interaction database and visualization tool
used for analyzing and interpreting protein-protein interactions based
on input protein sequences.

The analysis of the protein interactive network for AT3G15040 using the
STRING bioinformatics tool revealed interactions with proteins acting as
transcription factors and others involved in regulating lateral organ
growth, as well as stabilizing newly synthesized mitochondrial proteins.
